# nixduino
Build arduino sketches like any other nix derivation.
Look at `examples/` for how to use it.

## Supported Board Values:
- `uno`
- `leonardo`
- `mega`

## Supported Library Values:
- `Ethernet`
- `EEPROM`
- `Firmata`
- `GSM`
- `LiquidCrystal`
- `SD`
- `Servo`
- `SPI`
- `SoftwareSerial`
- `Stepper`
- `TFT`
- `WiFi`
- `Wire`
